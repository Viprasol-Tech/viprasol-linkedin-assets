### Post 2 (Afternoon) — EDUCATIONAL
```
How to read a backtesting report (most traders skip this):

PROFIT FACTOR (most important)
- Below 1.0: Losing strategy
- 1.0-1.5: Marginal (not worth automating)
- 1.5-2.5: Good
- Above 2.5: Excellent (or overfit — verify)

MAXIMUM DRAWDOWN
- Below 10%: Conservative
- 10-20%: Moderate
- 20-30%: Aggressive
- Above 30%: Dangerous

TOTAL TRADES
- Below 100: Not enough data (unreliable)
- 100-500: Decent sample
- 500+: Statistically significant

WIN RATE
- 40-50% can be profitable (if risk:reward is 1:2+)
- 60-70% is excellent
- 90%+ is probably overfit

SHARPE RATIO
- Below 1: Poor
- 1-2: Good
- Above 2: Excellent

Learn to read these numbers before investing in any EA.

Save this. You'll reference it often.

#backtesting #tradingmetrics #education #algotrading #data
```

```
- Free (you already own it)
```
